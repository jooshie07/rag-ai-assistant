\# 👨‍💻 Author



Built as a learning project to explore Retrieval-Augmented Generation (RAG), semantic search, and AI applications using Python. 

&#x20;#🤖 Pro RAG AI Assistant



A simple Retrieval-Augmented Generation (RAG) AI assistant built using Streamlit, FAISS, Sentence Transformers, and FLAN-T5.



The project retrieves relevant information using vector search and generates AI answers with source references.



\## ✨ Features



\- 🔍 Semantic search using FAISS

\- 🧠 Text embeddings with Sentence Transformers

\- 🤖 AI answer generation using FLAN-T5

\- 📚 Source display for retrieved context

\- 💻 Simple Streamlit web interface

\- ⚡ Lightweight and beginner-friendly RAG project

\## 🚀 Installation



Install dependencies:



```bash

pip install streamlit transformers sentence-transformers faiss-cpu torch

```



\## ▶️ Run the Project



Start the Streamlit app:



```bash

streamlit run app.py

```



Then open the browser and ask questions through the RAG assistant.
